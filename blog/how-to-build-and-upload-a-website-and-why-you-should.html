Build and Upload a Website.

April 7, 2022

Learn HTML and CSS basics

First of all, choose a good text editor, personally I use vim/neovim, It's available in linux, windows and mac, you can use VScode instead, but it's extremely bloated and slow.
Now, you have to see these 2 videos until the end!, LukeSmith's video (enough to build a simple personal website) and FreeCompCode video (Her voice is unbeatable lol).

After making the first web page, you have to make a blog, to make a blog you have many options like Hugo, Jekyll and many others. Personally I use lb blog system, It's a very lightweight 80 SLOC bash script, I have customized it to suit my website style, I like it but if you are a newbie, you may want to use something like Hugo or Jakyll.

Note: Know that you can take ideas from other websites, you can see HTML and CSS for any website in any browser.


Upload the website

You have to create a github account, then create a repo and name it "YOURWEBSITENAME.github.io" and check "Add a README file" box.
Now upload index.html and other website files you just made to the repo.
then type "YOURWEBSITENAME.github.io" in the url bar and boom! Your website is online now for free.

Note: if you didn't understand this part just type "Github Pages" in Youtube and you will find ton of videos about that, and remember to replace "YOURWEBSITENAME" with the name of your website.
